Flujo

Cómo funciona

Hablas con tu coach. Él pregunta, crea el plan y te lo abre en una Mini App. Tú entrenas; él ajusta en vivo.

1

Le dices que vas a entrenar

«Voy a entrenar». El coach pregunta energía, tiempo y molestias.

2

Crea el plan adaptado

Basado en tu perfil, historial y energía del día. Ejercicios, series y pesos sugeridos.

3

Te abre la Mini App

Un botón en el chat abre el plan visual. Ves cada ejercicio y registras tocando. ¿No convence? Lo descarta y crea otro.

4

Ajusta en vivo

¿Máquina ocupada? ¿Molestia? Se lo dices por chat y cambia el ejercicio al instante.

5

Feedback y memoria

Al terminar le cuentas cómo fue. Lo guarda todo y la próxima vez adapta mejor. Puedes compartir la rutina con un link de solo lectura.

Setup

Montar tu propio coach

Docker Compose en local o recursos independientes en Coolify. La aplicación es agnóstica al agente y el dominio es opcional.

1 — Levanta el stack

git clone https://github.com/jlfernandezfernandez/gym-tracker.git
cd gym-tracker
cp .env.example .env
docker compose up -d     # app:8000 + MCP:8001 + Postgres + MinIO

2 — Conecta tu agente

Endpoint MCP para Hermes, Claude, Codex u otro agente:
http://localhost:8001/mcp

URL pública opcional para la Mini App:
PUBLIC_APP_URL=https://gym.tu-dominio.com
CORS_ORIGINS=https://gym.tu-dominio.com

Guías: conectar un agente · desplegar en Coolify.

Stack

Simple a propósito

FastAPI + SQLModel

Backend async sobre Postgres. Auth HMAC de Telegram initData, scoping por usuario.

Mini App en Astro

Estática, TypeScript vanilla. Chart.js para progresión y body map anatómico como única isla React.

FastMCP

Herramientas por Streamable HTTP. Instructions + docstrings = documentación que el agente lee al conectarse.

Docker

API + Mini App en un contenedor. Compose local o Coolify en producción.
